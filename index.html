Introduction

Python is an interpreted, high-level, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant indentation.

This documentation will guide you through the basics of getting started with Python.

Installation

Before you can start writing Python code, you need to have Python installed on your system.

You can download the latest version of Python from the official Python website: python.org/downloads/

Follow the installation instructions for your specific operating system (Windows, macOS, Linux).

It's recommended to install pip, the package installer for Python, during the installation process.

Basic Syntax

Python syntax is known for its readability. Here are some fundamental syntax rules:

  • Indentation is crucial in Python. It's used to define code blocks.
  • Comments are used to explain code. Single-line comments start with a hash symbol (#).
  • Python is case-sensitive.

Example:

# This is a comment
print("Hello, Python!")

if 5 > 2:
    print("Five is greater than two!")
Variables and Data Types

Variables are used to store data. In Python, you don't need to explicitly declare the data type of a variable.

Common data types in Python include:

  • Integers (e.g., 10, -5)
  • Floats (e.g., 3.14, 2.5)
  • Strings (e.g., "Hello", 'World')
  • Booleans (e.g., True, False)
  • Lists (e.g., [1, 2, 3])
  • Tuples (e.g., (1, 2, 3))
  • Dictionaries (e.g., {"name": "Alice", "age": 30})

Example:

name = "Bob"
age = 25
is_student = True
grades = [85, 90, 92]
Operators

Python supports various operators to perform operations on values and variables.

Common operators include:

  • Arithmetic operators (+, -, *, /, %, **)
  • Comparison operators (==, !=, >, <, >=, <=)
  • Logical operators (and, or, not)
  • Assignment operators (=, +=, -=, etc.)

Example:

x = 10
y = 3
sum_result = x + y
greater_than = x > y
logical_and = x > 5 and y < 5
Control Flow

Control flow statements determine the order in which code is executed.

Key control flow statements in Python are:

  • if, elif, else statements for conditional execution.
  • for loops for iterating over sequences.
  • while loops for repeating a block of code as long as a condition is true.

Example:

age = 18
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

for i in range(5):
    print(i)

count = 0
while count < 3:
    print(count)
    count += 1
Functions

Functions are blocks of reusable code that perform a specific task. You can define your own functions or use built-in functions.

Functions are defined using the def keyword.

Example:

def greet(name):
    print(f"Hello, {name}!")

greet("Alice")

def add(a, b):
    return a + b

result = add(5, 2)
print(result)
Common Libraries

Python has a vast ecosystem of libraries that extend its functionality. Here are a few common ones:

  • NumPy: For numerical computations.
  • Pandas: For data manipulation and analysis.
  • Matplotlib: For creating visualizations.
  • Requests: For making HTTP requests.
  • Django and Flask: For web development.

You can install libraries using pip:

pip install numpy
